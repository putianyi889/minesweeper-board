Minesweeper artwork used by the board components.

Background assets:

- `celldown.svg` for opened/down cells
- `cellup.svg` for unopened/up cells

Foreground assets:

- `null.svg` for blank cells
- `num1.svg` ... `num8.svg` for numbers
- `mine.svg` for a mine
- `flag.svg` for a flag
- `falsemine.svg` for a wrongly marked mine
- `blast.svg` for the exploded mine

Counter assets:

- `counter/counter-.svg` for minus signs
- `counter/counter0.svg` ... `counter/counter9.svg` for digits
